
# Smart Bookmark App - Premium Edition

Features:
- Google OAuth (Supabase)
- Private bookmarks (RLS)
- Real-time sync
- Animated premium UI (Framer Motion)
- Glass + gradient neon design
- Production-ready for Vercel

Setup:
1. Create Supabase project
2. Enable Google OAuth
3. Create bookmarks table with RLS
4. Add env vars in Vercel
5. Deploy
